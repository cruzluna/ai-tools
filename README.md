# ade.codes

A collection of ai tools, agent development environments, and agent frameworks.
I see so many different ai and agent related tools popping up and I wanted to 
codify a list. 

If you want to add another or even another filter, submit an issue as a prompt request and 
I will have one of my agents get to it.

## Setup

```bash
bun install
```

## Run

```bash
bun run dev
```

## Build

```bash
bun run build
```

## Lint

```bash
bun run lint
```
